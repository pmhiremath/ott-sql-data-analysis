
# OTT Platform Data Analysis (SQL Project)

## Project Overview
This project analyzes an OTT streaming platform dataset using SQL to understand user behavior, subscriptions, and engagement.

## Tools Used
- SQL
- MySQL
- Data Analysis

## Dataset Tables
- users
- plans
- subscriptions
- content
- watch_history
- ratings

## Key Analysis
1️⃣ Active Users per Subscription Plan
2️⃣ Monthly Revenue by Subscription Plan
3️⃣ Active Subscribers Who Never Watched Content
4️⃣ Inactive Users (No Activity in Last 30 Days)
5️⃣ Average Watch Time per Genre
6️⃣ Content Completion Rate
7️⃣ Top Power Users by Total Watch Time
8️⃣ Most Watched Genre by Country
9️⃣ High-Rated but Low-Engagement Content
🔟 Customer Churn Rate Analysis
1️⃣1️⃣ Revenue Lost Due to Subscription Cancellations
1️⃣2️⃣ Subscription Plan Retention Rate
1️⃣3️⃣ Plan Upgrade and Downgrade Analysis
